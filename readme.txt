WordPress multipurpose form's builder, store data in database, and receive notifications via email. Export the submitted data from the admin panel to CSV format.

== Description ==

Easily create and manage forms through the WordPress admin panel. Tiger Forms has powerful and reliable step by step jQuery form validation. Have all standard input fields with lots of features. Manage submitted data easily on WordPress admin panel. There is another special feature is you can easily export all submitted data tables to CSV format. Support multiple multi-step forms on the same page. Mobile friendly, fully responsive design, cross browser compatible.

== Core Features ==

* Easy to create Contact/Data form.
* Drag and Drop form builder.
* Insert data to the database.
* Create an input field with unique data validation.
* Get Email notification when data was submitted.
* Manage submitted data easily on WordPress admin panel.
* Export submitted data on the WordPress admin panel to CSV format.
* Faster data submission from frontend.
* Jquery form validation and data submission.
* You can add custom CSS classes that will help you create custom form designs.

== Installation ==

* Method#1:
1. Navigate to: Plugins -> Add new -> Search.
2. Search for "Tiger Forms - Drag and Drop Form Builder".
3. Install and activate plugin.
4. Click the plugin page link or navigate to: Tiger Forms Menu.

* Method#2:
1. Download plugin (zip file).
2. Upload and install.
3. Activate plugin.
4. Click the plugin page link or navigate to: Tiger Forms Menu.
